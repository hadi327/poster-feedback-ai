# 🎨 Poster Feedback AI

A lightweight, interactive web application built with Python and Streamlit designed to analyze poster compositions, layouts, and image metadata to provide instant, structured design critiques.

---

## ✨ Features
* 📁 **Drag-and-Drop Upload:** Supports `.png`, `.jpg`, and `.jpeg` poster designs.
* 📊 **Instant Dashboard UI:** Side-by-side view displaying the uploaded asset alongside an automated assessment.
* 📐 **Smart Layout Parsing:** Uses image dimensions and metadata to dynamically evaluate structural metrics.
* 📝 **Categorized Critiques:** Generates constructive scores, observations, and actionable advice across 4 foundational pillars:
  * Visual Hierarchy
  * Typography
  * Color Palette
  * Composition & Layout
